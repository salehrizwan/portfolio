Second Project

SwiftRate is a modern and responsive currency converter web application that allows users to quickly and accurately convert between 160+ global currencies using real-time exchange rate data from an external API. It provides an easy-to-use interface where users can enter an amount, select source and target currencies, and instantly view the converted value along with the live exchange rate. The application also includes a swap feature to quickly interchange currencies and dynamically updates results as the user interacts with the inputs. Built using HTML, CSS, and JavaScript with Fetch API integration, SwiftRate is designed to demonstrate real-world API usage, asynchronous programming, and practical frontend development skills in a simple yet effective fintech-style project.
